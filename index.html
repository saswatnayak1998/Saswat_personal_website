    Design Principles of 3D Photonic Crystals

    We make 2 descriptors for 3D photonic crystals, one based on local environment, and one based on topology. Analyzing the ML featurizations for these photonic structures, we use the similarities within these feature spaces as a proxy for similarity between these crystals, and derive an "atlas" for photonic crystal design. Furthermore, we extract insights from these representations to guide self-assembly processes towards scalable, robust photonics manufacturing.
